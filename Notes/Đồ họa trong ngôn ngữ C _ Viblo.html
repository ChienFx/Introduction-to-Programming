      Giới thiệu

      Khi viết 1 chương trình thì ngoài những tiện ích mà nó mang lại, yếu tố đồ họa của chương trình đó cũng không kém phần quan trọng giúp chương trình dễ tiếp cận với người dùng.

      Với các chương trình của ngôn ngữ lập trình C cũng vậy. Mặc dù ít được nhắc tới, tuy nhiên, C cũng hỗ trợ các hàm đồ họa nằm trong thư viện graphics.h.

      Sau đây, chúng ta sẽ tìm hiểu về cách cài đặt thư viện graphics và các hàm đồ họa khả dụng trong C cũng như 1 số ví dụ của chúng.

      I. Cài đặt thư viện graphics trong C

      a. Cài đặt với trình biên dịch Dev-C++ trong Window

      • Download và cài đặt Dev-C++
      • Download thư viện graphics.h lưu vào thư mục con include/ của thư mục Dev-C++ bạn đã cài đặt.
      • Download thư viện libbgi.a lưu vào thư mục con lib/ của thư mục Dev-C++.
      • Tạo 1 new project để thực hành thao tác đồ họa trong C, ví dụ GraphicProject.

      • Trên thanh menu, click vào Project->Project Option. Chọn tab Parameters. Trong khung textbox Linker, copy và paste đoạn sau:

        -lbgi
-lgdi32
-lcomdlg32
-luuid
-loleaut32
-lole32

      • Click OK.

      b. Cài đặt trong linux

      • Cài đặt các gói thư viện cần thiết
      sudo apt-get update

sudo apt-get install libsdl-image1.2 libsdl-image1.2-dev guile-1.8 guile-1.8-dev libsdl1.2debian libart-2.0-dev libaudiofile-dev libesd0-dev libdirectfb-dev libdirectfb-extra libfreetype6-dev libxext-dev x11proto-xext-dev libfreetype6 libaa1 libaa1-dev libslang2-dev libasound2 libasound2-dev

      • Download libgraph

      • Copy file libgraph-1.0.2.tar.gz vào thư mục home rồi giải nén, sau đó chạy tiếp các lệnh sau:
      cd libgraph-1.0.2
./configure
sudo make
sudo make install
sudo cp /usr/local/lib/libgraph.* /usr/lib
      • Cài đặt trình biên dịch C, ví dụ Geany (tương đối giống với Dev-C++).

      Vậy là bước cài đặt để sử dụng thư viện graphics đã hoàn tất.

      II. Các hàm đồ họa của C

      a. initgraph

      • Cấu trúc:

      void initgraph(int *graphdriver, int *graphmode, char *pathtodriver);

      • Mô tả:

      Sử dụng để khởi tạo hệ thống đồ họa. Trong đó:

      Tham số Mô tả
      pathtodriver Đường dẫn đến thư mục chứa các driver đồ hoạ
      graphdriver Card màn hình
      graphmode Mode màn hình

      Tham số thứ 3 với Dev-C++ là: C:\\TC\\BGI, còn với trình biên dịch trong Linux là NULL.

      b. closegraph

      • Cấu trúc:

      void closegraph();

      • Mô tả:

      Hàm closegraph có chức năng đóng chế độ đồ họa, giải phóng tất cả bộ nhớ cấp phát bởi hệ thống đồ họa và phục hồi màn hình sang chế độ như trước khi gọi hàm initgraph.

      • Ví dụ:
      #include&lt;graphics.h&gt;
#include&lt;conio.h&gt;

int main()
{
   int gd = DETECT, gm;

   initgraph(&amp;gd, &amp;gm, &quot;C:\\TC\\BGI&quot;);
   // TO DO (h&agrave;m đồ họa)
   getch();
   closegraph();
   return 0;
}

      c. arc

      • Cấu trúc:

      void arc(int x, int y, int stangle, int endangle, int radius)

      • Mô tả:

      Được sử dụng để vẽ 1 vòng cung với hàm tâm có tọa độ (x, y), góc bắt đầu stangle, góc cuối endangle và bán kính cung radius. Hàm này cũng có thể được sử dụng để vẽ 1 hình tròn rỗng hoàn chỉnh nếu tổng góc bắt đầu và góc cuối là 360.

      • Ví dụ:
      #include &lt;graphics.h&gt;
#include &lt;conio.h&gt;

int main() {
   int gd = DETECT, gm;

   initgraph(&amp;gd, &amp;gm, &quot;C:\\TC\\BGI&quot;);

   arc(100, 100, 0, 180, 100);
   arc(305, 100, -90, 180, 100);
   arc(510, 100, 0, 360, 100);

   getch();
   closegraph();
   return 0;
}

      d. bar

      • Cấu trúc:

      void bar(int left, int top, int right, int bottom);

      • Mô tả:

      Hàm bar được sử dụng để vẽ 1 hình chữ nhật đặc 2 chiều. Các tham số left, top, right, bottom dùng để xác định tọa độ điểm trái trên và điểm phải dưới của hình chữ nhật.

      • Ví dụ:
      #include &lt;graphics.h&gt;
#include &lt;conio.h&gt;

int main()
{
   int gd = DETECT, gm;

   initgraph(&amp;gd, &amp;gm, &quot;C:\\TC\\BGI&quot;);

   closegraph();
   return 0;
}

      e. bar3d

      • Cấu trúc:

      void bar3d(int left, int top, int right, int bottom, int depth, int topflag);

      • Mô tả:

      Hàm bar3d dùng để vẽ hình chữ nhật 3 chiều. Cũng tương tự với hàm bar, hình chữ nhật 3 chiều có các tham số left, top, right, bottom để xác định tọa độ điểm trái trên và điểm phải dưới. Ngoài ra, còn có tham số depth xác định chiều cao và top để xác định xem hình chữ nhật có nắp hay không.

      g. drawpoly

      • Cấu trúc:

      void drawpoly( int num, int *polypoints );

      • Mô tả:

      Dùng để vẽ đa giác rỗng. Với đa giác n đỉnh thì tham số num truyền vào phải là num = (n+1) và mảng chứa tọa độ các đỉnh polypoints sẽ cần truyền vào (n+1) cặp tọa độ đỉnh. Lưu ý, tọa độ đỉnh cuối cùng sẽ bằng với tọa độ của đỉnh đầu tiên.

      • Ví dụ:
      #include &lt;graphics.h&gt;
#include &lt;conio.h&gt;

{
   int gd = DETECT, gm, points[] = {320, 150, 420, 300, 250, 300, 320, 150};

   initgraph(&amp;gd, &amp;gm, &quot;C:\\TC\\BGI&quot;);
   // Vẽ tam gi&aacute;c
   drawpoly(4, points);

   getch();
   closegraph();
   return 0;
}

      h. fillpoly

      void fillpoly( int num, int *polypoints );

      • Mô tả:

      Dùng để vẽ đa giác đặc, các tham số y hệt với hàm vẽ đa giác rỗng drawpoly được mô tả ở trên.

      • Ví dụ:

      i. ellipse

      Vẽ 1 elip rỗng với tọa độ tâm (x, y), góc bắt đầu stangle, góc kết thúc endangle và 2 bán kính xradius, yradius.

      j. fillellipse

      • Cấu trúc:

      void fillellipse(int x, int y, int xradius, int yradius);

      • Mô tả:

      Vẽ elip đặc với tọa độ tâm (x, y) và 2 bán kính xradius, yradius tương ứng với 2 tọa độ x, y.

      • Ví dụ:

#include &lt;conio.h&gt;

      • Output:

      k. floodfill

      • Cấu trúc:

      void floodfill(int x, int y, int border);

      • Mô tả:

      Tô màu 1 hình kín chứa điểm x, y và border của hình kín có màu là border.

      Hàm floodfill không những phụ thuộc vào các tham số trên, mà còn phụ thuộc vào trạng thái màn hình.

      Nếu hình kín có mã màu bằng giá trị của border thì :

      • Nếu điểm có tọa độ (x, y) nằm trong hình kín thì hình kín được tô màu.

      • Nếu điểm có tọa độ (x, y) nằm ngoài hình kín thì phần bên ngoài hình kín được tô màu.

      Nếu màn hình không có hình kín như trên thì cả màn hình sẽ được tô màu.

      • Ví dụ:
      #include &lt;graphics.h&gt;
#include &lt;conio.h&gt;

int main()
{

   initgraph(&amp;gd, &amp;gm, &quot;C:\\TC\\BGI&quot;);

   circle(100, 100, 50);
   floodfill(100, 100, RED);

   getch();
   closegraph();
   return 0;
}
      • Output:

      l. getarccords

      • Cấu trúc:

      void getarccoords(struct arccoordstype *var);

      • Mô tả:

      Hàm này sử dụng để lấy tọa độ của vòng cung được vẽ gần đây nhất. Trong đó, arccoordstype được định nghĩa như sau:

      struct arccoordstype
{
   int x, y;                /*   tọa độ t&acirc;m của v&ograve;ng cung  */
   int xstart, ystart;      /*   vị tr&iacute; bắt đầu            */
   int xend, yend;          /*   vị tr&iacute; kết th&uacute;c           */
};
      • Ví dụ:
      #include&lt;graphics.h&gt;
#include&lt;conio.h&gt;
#include&lt;stdio.h&gt;

int main()
{
   int gd = DETECT, gm;
   struct arccoordstype a;
   char arr[100];

   initgraph(&amp;gd, &amp;gm,&quot;C:\\TC\\BGI&quot;);

   arc(250, 200, 0, 90, 100);
   getarccoords(&amp;a);

   sprintf(arr, &quot;(%d, %d)&quot;, a.xstart, a.ystart);
   outtextxy(360, 195, arr);

   sprintf(arr, &quot;(%d, %d)&quot;, a.xend, a.yend);
   outtextxy(245, 85, arr);

   getch();
   closegraph();
   return 0;
}
